# Data Professional Survey Dashboard

This project is an interactive Power BI dashboard built using survey data from data professionals.

## Dashboard Features
- Average salary analysis
- Work-life balance insights
- Happiness with coworkers
- Favorite programming languages
- Country-wise breakdown
- Role-based salary comparison

## Tools Used
- Power BI
- Power Query
- DAX
